# Caso práctico – Análisis de Datos Deportivos

Este repositorio contiene el diseño de un sistema inteligente de apoyo a la toma de decisiones ante la congestión competitiva en fútbol profesional.

## Objetivo del proyecto
Analizar el impacto de la congestión de partidos en el rendimiento físico y técnico de los jugadores y plantear un sistema aplicado de apoyo a la toma de decisiones para el cuerpo técnico.

## Estructura del repositorio
- data/: datos del proyecto (no incluidos)
- scripts/: scripts de limpieza, transformación y análisis
- reports/: informe del caso práctico (Word y Quarto)
- outputs/: resultados y visualizaciones
- docs/: documentación adicional

## Descripción
El proyecto propone un enfoque aplicado de análisis de datos deportivos, incorporando herramientas de análisis estadístico, visualización y modelos de lenguaje (LLMs) para la generación automática de informes.
